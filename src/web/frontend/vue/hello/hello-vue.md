## 什么是 Vue

Vue (发音为 /vjuː/，类似 view) 是一款用于构建用户界面[^1]的渐进式框架。它基于标准 HTML、CSS 和 JavaScript 构建，并提供了一套**声明式**的、**组件化**的**编程模型**，帮助你高效地开发用户界面。无论是简单还是复杂的界面，Vue 都可以胜任。
